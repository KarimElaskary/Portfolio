# Karim Elaskary - Personal Portfolio

A modern, responsive, and highly interactive personal portfolio website designed to showcase my projects, skills, and experience as a Front-End Developer. Built with a focus on smooth animations, clean typography, and a dynamic user experience.

## ✨ Features

- **Modern UI/UX**: Designed with a sleek, dark-themed aesthetic featuring vibrant accents and glassmorphism elements.
- **Interactive Animations**: Implemented smooth scroll, page transitions, and element reveals using AOS (Animate on Scroll) and WebGL-based effects (OGL).
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewports ensuring a seamless experience across all devices.
- **Dynamic Projects Showcase**: Features an interactive projects section utilizing `Swiper.js` for an intuitive sliding gallery of featured work.
- **Interactive Contact Section**: Direct email links, social media integration, and a clean, accessible layout for professional inquiries.
- **Particle Effects**: Engaging background particle animations built to enhance the visual depth of the landing page.

## 🛠️ Tech Stack

This project was built using the following modern web technologies:

- **Framework:** [React 18](https://react.dev/) + [Vite](https://vitejs.dev/) for blazing-fast development and optimized production builds.
- **Routing:** [React Router DOM](https://reactrouter.com/) for seamless, client-side page navigation.
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) for utility-first, rapid UI development and custom design tokens.
- **Animations & Effects:**
  - [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) for smooth element reveals.
  - [OGL](https://github.com/oframe/ogl) for lightweight WebGL rendering and particle effects.
  - [Swiper](https://swiperjs.com/) for touch-enabled, responsive sliders.
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/) for scalable vector graphics.

## 🚀 Running the Project Locally

To get a local copy up and running, follow these simple steps:

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn package manager

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Visit `http://localhost:5173` to view the application.

## 📁 Project Structure

- `src/components/`: Contains reusable UI components (`Hero`, `AboutMe`, `MyWork`, `Header`, `Footer`, `Particles`, etc.)
- `src/assets/`: Static assets including images, icons, and mockups.
- `src/AllProjects.jsx`: The dedicated page showcasing a comprehensive list of all projects.
- `src/Home.jsx`: The main landing page aggregating the primary sections.
